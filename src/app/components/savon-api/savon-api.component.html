Présentation

Contexte

Le projet est un partenariat entre l’entreprise Adepro (prestataire pour la mise sur le marché des produits cosmétiques) et le lycée Léonard de Vinci de Melun.

  • Site Adepro
  • Site du lycée LDV

Adepro souhaite créer une application de type simulateur/calculateur qui permet aux utilisateurs de saisir une recette de savon et d’obtenir les scores sur les différentes caractéristiques du savon (douceur, séchage, pouvoir lavant, etc.).

Une recette de savon est constituée de :

  • Un ou plusieurs corps gras (ex : huile d’olive, huile de noix de coco, etc.)
  • Un agent alcalin (soude ou potasse)
  • Éventuellement des ingrédients supplémentaires (ex : cire d’abeille)

Il y a deux demandes spécifiques :

  • Pour chaque caractéristique du savon en plus d’un simple de score ilfaut une « mention » exemple : un savon qui un score de douceur entre 4 et 5 est doux un savon entre 1 et 2 est dur …
  • Les caractéristiques du savon doivent être affiché sous la forme d’undiagramme Radar
